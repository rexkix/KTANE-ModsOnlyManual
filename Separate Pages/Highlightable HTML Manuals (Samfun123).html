On the Subject of Highlightable HTML Manuals

Oh, Wow! I didn't even know you could make it do that.

To install:
  1. Add the Tampermonkey extension to your browser:
    • Google Chrome
    • Firefox
    • Safari
    • Microsoft Edge
    • Opera Next
    • A full list can be found here.
  2. Install the userscript by clicking here.
  3. You will now be able to highlight HTML manuals from ktane.timwi.de.
  4. Some browsers (like Google Chrome) require that you give extentions permission to access to file urls. You must give Tampermonkey that access if you want to use locally saved html manuals.
Controls:
  • Click Highlights a cell in a table or line of text.
  • Ctrl + Click Highlight a row in a table.
  • Alt + Click Highlight a column in a table.
  • Shift + T to toggle the highlighting.
